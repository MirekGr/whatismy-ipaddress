# WhatIsMyIPAddress.com

Free online privacy and network diagnostic toolkit.

🔍 **Live tool:** https://whatismy-ipaddress.com

## Features
  - IP Address lookup (IPv4 & IPv6)
  - DNS Leak Test
  - WebRTC Leak Test
  - Port Scanner
  - Password Breach Check
  - Email Breach Check
  - VPN / Proxy Detection
  - Browser Fingerprint Analysis
  - Internet Speed Test

## Privacy
  All checks run client-side where possible. No data is stored or logged.
